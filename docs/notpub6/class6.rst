F5 Distributed Cloud 212 - Advanced Client Side Defense
=======================================================

F5 Distributed Cloud's SaaS Client Side Defense can help you mitigate application threats. During this lab you can use your own F5 Distributed Cloud account
and your own application (origin server) that you are going to protect with CSD. Alternatively you can also use the F5 sales-demo application https://shop.sales-demo.f5demos.com
where the required JavaScript tag for CSD has already been added.
